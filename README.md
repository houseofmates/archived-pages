# archived-pages

a browser extension that checks archived versions of web pages when a page is unavailable.

## what is this?

a browser extension that checks archived versions of web pages when a page is unavailable.

this is a fork of [dessant/web-archives](https://github.com/dessant/web-archives) with a redesigned ui.

## original vs. this version

this is a fork of the [dessant/web-archives](https://github.com/dessant/web-archives) project.

### changes from original

- redesigned the ui with a dark pkm aesthetic. all text is now lowercase. extension renamed from archived-pages to archived-pages. auto-update checking added.

## installation

download the `.xpi` file from releases and open it in firefox, or load it temporarily via `about:debugging` > this firefox > load temporary add-on.
